What is Ext.js?

Sencha Ext.js

Javascript Framework

ExtJS is a javascript framework developed by Sencha.It provides extensive control over application, a developer needs, to build data-intensive, cross-platform web applications. Ext JS leverages HTML5 features on modern browsers. Ext JS features 115+ high-performance, pre-tested and integrated UI components including calendar, grids, charts and more. The Ext JS Grid and Advanced Charting package can handle millions of records with ease. ExtJS has been a popular choice when the need is to build single page web application or handling tons of data from server. Ext's Grid component can beautifully handle data and providing complete control over it.

Features of Ext.js are:

  • Reduces the cost of web application development.
  • Customizable UI widgets with collection of rich UI such as grids, pivot grids, forms, charts, trees.
  • A flexible layout manager helps to organize the display of data and content across multiple browsers, devices, and screen sizes.
  • Follows MVC architecture, hence the code is highly readable.
