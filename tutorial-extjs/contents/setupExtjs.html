Setting up Ext.js?

Sencha Ext.js

Javascript Framework

Setting up Ext.js is very easy, similar to any javascript library. It can be set up in two ways. Let's discuss them in turns.

  1. Using Sencha Ext.js SDK

    Sencha Ext.js SDK can be download from their website. The download SDK contains all the files, themes and almost everything that you would ever require to build an Extjs application. It comes at a trial of 30 days. Moreover, Secha also provides GPL licensing for open source development which is available here. User can start building application on this SDK.

  2. Using CDN

    CDN is content delivery network with which you do not need to download the Ext JS library files, instead you can directly add CDN link for ExtJS to your program
